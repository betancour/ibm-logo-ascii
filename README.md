# IBM ASCII Logo Experiment

This repository contains an ASCII and Unicode (Braille-based) text-mode
representation of the IBM logo.

The project is a **technical and artistic experiment** focused on:
- ASCII art
- Unicode Braille rendering
- Text-mode graphics
- Terminal-based visual techniques inspired by historical computing

Text-mode representations of well-known logos have long been used in
computing culture for learning and experimentation, particularly on
systems with limited or no graphical capabilities.

This project follows that tradition in a transparent and respectful way.
